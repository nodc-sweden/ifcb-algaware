# algaware: Interactive IFCB Phytoplankton Reporting Tool

A Shiny application for processing, validating, and reporting
phytoplankton data from Imaging FlowCytobot (IFCB) instruments.
Integrates with IFCB Dashboard for data retrieval, supports interactive
validation of classifier predictions, and generates Word reports with
biomass plots, heatmaps, and image mosaics.
