# Chlorophyll wording for prompts and instructions

Chlorophyll wording for prompts and instructions

## Usage

``` r
chl_measure_terms(chl_measure = "fluorescence")
```

## Arguments

- chl_measure:

  Either `"fluorescence"` (FerryBox/CTD) or `"concentration"` (LIMS
  bottle/hose filter samples).

## Value

A list with `data_label` (prompt data line), and English (`en`) and
Swedish (`sv`) terms for the measurement.
