# Data Loader Module UI

Sidebar controls for selecting a cruise or date range and loading data.

## Usage

``` r
mod_data_loader_ui(id)
```

## Arguments

- id:

  Module namespace ID. Shiny modules use namespaced IDs so that multiple
  instances of the same module don't conflict. `NS(id)` creates a
  function that prefixes all input/output IDs with this namespace.

## Value

A UI element.
